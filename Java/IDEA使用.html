  1. 快捷键设置
            Keymap: 可以设置某一种软件或系统的风格，转换了后可能部分快捷键无法使用
Mac 来说 ctrl 指的是 command 键
                 alt  指的是 option 键
1. main              // public static void main(String[] args) {}
2. sout              // System.out.println();
3. ctrl + D          // 复制选中，未选中则复制当前行
4. ctrl + shift 
   + ⬆️/⬇️          // 移动代码行的下上位置
   + ⬅️/➡️          // 当前光标向左或右全选中

5. ctrl + /         // 单行注释，但选中多行，此快捷也是多行的每行的单行注释
6. ctrl + shift + / // 多行注释，指 /*…*/
7. ctrl + shift + t // 提示是否创建 test class

*7. ctrl + alt + t   // 对选中的代码区域增加 try catch 或其它语法块，是可选择的 
*8. alt + enter      // 可以对选中的类进行导包，和对new的类生成变量

*9. ctrl + alt + L   // 格式化代码
10. ctrl + 1/2/3/4   // 可以展开缩进不同的工作栏，常见：1-项目结构

debug模式下
1. F8 折线蓝色箭头：单步执行，会把方法当做一行代码执行，不会进入方法
2. F7 向下蓝色箭头：单步执行，但会进入自定义方法，不会进入jdk源码的方法
3. 向下红色箭头：单步执行，会进入所有方法（包括jdk源码）
4. 向上蓝色箭头：跳出当前所在的方法
5. 斜着蓝色箭头：执行到光标所在的行
6. 左竖栏绿色三角：执行到下一个断点
7. 红圆圈斜线：所有断点失效
8. 双重叠红圆圈：断点管理窗口（可以删除指定断点）
*9. 条件断点，在红色断点处右键，会弹出条件设置框，当条件满足才会进入断点
*10.Evaluate计算值，监听某个变量，则在 右下角输出界面 variables + 一个变量名进行 watch!!!
