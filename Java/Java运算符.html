二元运算规则
1. 整数运算: 如果两个操作数有一个是long，结果也是long
            如果没有long，结果为int，则不管操作数是 byte 还是 short，结果也是int

    31 / 3 结果为int，因为整数常量默认为int，则31与3都是int，则结果也是int，且会把除后的小数干掉

2. 浮点运算: 如果两个操作数有一个是double，结果也是doouble
            只有两个操作数都是float，结果才是float
    如果有整数与浮点运算以大范围为准（一般来说是double）

3. 取模运算: 余数符号与左边操作符一致
4. char类型: 可以 ++ —- ，但是将unicode进行自增与自减，而不是拼接
扩展运算符：a *= b + 3 右边是一个整体  a = a * (b + 3)
关系运算符：
== !=      // 所有数据类型都可以使用
> >= < <=  // 仅针对数值类型与char类型
逻辑运算符
& &&   // 与，短路与
| ||   // 或，短路或
!      // 非，取反
^      // 异或，相同为false，不同为true  a^b
逻辑非 > 逻辑与 > 逻辑或
位运算
<<     // 左移运算符，左移1位相当于*2，2位相当于*4
>>     // 右移运算符，右移1位相当于/2 取商
