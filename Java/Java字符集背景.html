1. ASCII字符，只用1个字节，8位 — 可表示 256个 字符，英文/数字/特殊字符/控制符 都可以表示
  2. ISO8859-1 西欧字符集，把 256个字符全用完了
  3. 中国汉字太多，表示不完，所以采用2个字节，2^16 = 65535
  4. 最开始国标码：GB2312 — 扩展 GBK — GB18030（后面想把藏文，少数民族弄进来）
  5. 国际化：Unicode字符集（也是两个字节表示一个字符）
衍生出3种编码规范（字符集还是2个字节，但编码规范不一样）
1. UTF-8编码（主流编码规范）：[1个字节 - 英文]  [2个字节 - 其它文字]  [3个字节 - 汉字]
2. UTF-16编码
3. UTF-32编码
Java中 char类型采用 Unicode 字符集
char c1 = ‘\u0061’;            // 表示字符Unicode 61 为 a
转义字符：char c2 = ‘\n’  // 表示换行

Java基本类型中没有字符串类型，String字符串是一个类
