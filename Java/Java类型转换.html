1.自动转换
容量小的数据类型可以自动转换为容量大的数据类型（容量指这个类型表示的范围）
byte -> short -> int -> long   char -> int [因为char是2个字节，int是4个字节，对应的范围也是小到大，所以也可以转]
float -> double  int -> double [int是4个字节，double是8个字节同上]
上面的自动转换无数据丢失
int  -> float
long -> float
long -> double
上面3个可能会有精度的损失
特例：
整数常量是int类型，可以自动转成 byte/short/char [只要不超过对应类型的表示范围]；如 byte h1 = 123; char h3 = 97
2.强制类型转换（造型，cast）
(type)var 强制转换数值的类型，可能损失精度
double a = 3.98234;
int b = (int)a;    // 3
int c = 97;
char d = (char)c;  // a
强制转型，超过了表数范围，则会转成一个完全不同的值
byte e = (byte)300;  // 44, 有规律不用研究
