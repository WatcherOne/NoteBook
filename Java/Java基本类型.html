局部变量：方法或语句块内部
成员变量（实例变量）：类内部，方法外部，对象创建时，也跟着创建，对象消失时，也跟着消失
静态变量（类变量）：类内部，static修饰，类，类被加载静态变量就有效，被卸载就消失
  1. final 来定义一个常量，一被初始化后就不能更改，常量命名全用大写，单词之间_连接

1.最前面一位表示符号位，以补码表示（原码，反码）
二进制的位数=2  则 2^2 表示 4种 组合【0,1,2,3】, 而最大值为 2^2 - 1 = 3, 因为4种组合包含了一个0
以补码表示，则 -0 就不存在，则 -0 就代表了 -128 [ 10000000 ]
2.十进制，二进制，八进制，十六进制表示方式差不多
3.浮点类存在舍入误差，数字不能精确表示，适合普通的科学和工程计算，精度足够；
   但不适合精度要求高的商业计算，这时候需要用 BigDecimal 进行运算和比较
4.整数常量 30000 这种，默认类型是 int 如果本身常量超过 int 范围则 加L 表示 long 类型，只是指整数常量为long类型
                   如：long a = 300000000L;  float a = 300000000L;  double a = 30000000L
   浮点常量 3.14 这种，默认类型是 double 如果要设置给 float 必须 加F 表示 float 类型
